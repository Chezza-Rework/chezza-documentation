# LockInv

:::note
ONLY clientside
:::

<span style={{color: 'red'}}>**Parameters**</span>\
**toggle** - `boolean <true/false>` - Lock/Unlock the Inventory

```lua
exports.inventory:LockInv(true) -- Locks the Inventory
exports.inventory:LockInv(false) -- Unlocks the Inventory
```
